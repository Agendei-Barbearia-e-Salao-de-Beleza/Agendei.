# Skill: `/speckit.checklist`

**Objetivo:** Validar a qualidade dos requisitos por meio de uma matriz de conformidade.

**Ação do Agente (Claude Code):**
1. Gere uma tabela de verificação (Checklist) com base na especificação.
2. Confirme ativamente se todas as regras de negócios, contratos de API e segurança (ex: RLS, validações) foram abordados no documento da Fase 2 (`spec/2_requirements/`).
3. Se algo estiver faltando, reporte imediatamente e pause o processo.
